# TaskManagement Project
This is a SpringBoot Application project desgined for managing tasks supporting standard CRUD (Create,Read,Update,Delete) operations.

##Requirements
1. Java Development Kit (JDK) - 17 or higher
2. Maven - 3.6.3 or higher
3. Git

##Installation and Project Setup
Please follow the below steps to run the project locally in your machine
1. Clone the Repository
   Open your command prompt and run the following command to clone the project
   git clone <HTTPS Repository URL>
   
   Once the Repository is cloned,navigate inside the project by the following command
   cd TaskManagement
   
2. Install Maven Dependencies
   Once Navigated inside the project (/TaskManagement), use Maven to install dependencies by the following command
   mvn clean install

3. Running the Application
   Run the project by using the SpringBoot Maven Plugin to start the application
   mvn spring-boot:run

   The application will start on port 8080 by default.

##API Endpoints and description
  The application is launched using Tomcat server with the base URL - http://localhost:8080/tasks. You can use Postman to test the API endpoints.
   
   Method  Endpoint    Description
   POST   /tasks       Create a new Task
   GET    /tasks/{id}  Retrieves a single task by ID
   PUT    /tasks/{id}   Updates a single task by ID
   DELETE /tasks/{id}   Deletes a single task by ID
   GET    /tasks       Retrieves all the tasks

   The parameters used in the task are:
   
   Parameters  Type
   id          String (Autogenerated)
   title       String (Mandatory field)
   description String (Optional field)
   status      IN_PROGRESS,PENDING (default),DONE
   due_date    Date (Mandatory field)

   Request Body Example (for Postman testing):
   {
    "title": "This is title", 
    "description": "This is description", 
    "status": "PENDING",    
    "due_date": "2025-11-23"
   }
